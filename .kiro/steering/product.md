# Product Overview

This is a hackathon project that combines a React frontend with AWS Bedrock AI integration. The application appears to be an AI-powered web application that leverages Amazon's Nova Lite model for conversational AI capabilities.

## Key Features
- React-based web interface
- AWS Bedrock integration for AI/ML functionality
- Educational AI assistant focused on artificial intelligence and machine learning topics

## Target Use Case
The application serves as an educational tool, designed to explain complex AI/ML concepts (like convolutional neural networks) to students in an accessible way.
